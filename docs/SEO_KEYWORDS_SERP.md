# MailKit — Long-tail SEO keywords + SERP-чек (2026-07-04)

Углы позиционирования, под которые эти ключи (после разбора конкурента SendMailAs):
1. Несколько ящиков/доменов, плати по факту, без возни, всё в одном Gmail.
2. One-time вместо годовой подписки.
3. Легитимный 1:1 объём (занятой консультант/поддержка/фрилансер, кто много отвечает).

**Cold outreach — исключён** (против Postmark ToS §5: unsolicited + purchased lists запрещены; общий сервер → суспенд у всех).

Точные объём/конкуренция/бид — снимать в **GKP (Boris)**. Ниже — сиды + что реально в выдаче на 2026-07-04.

## SERP-чек по коммерческим ключам (по факту выдачи 2026-07-04)

- **C «without Google Workspace»** — топ забит МЕЛКИМИ сервисами-конкурентами (не гиганты): MailBridge, Shipmail, Folio, Mailsuite, Kivit + блоги. Ранкабельно (нет Google/Zapier-стены), НО ниша ПЛОТНЕЕ, чем «только SendMailAs» — тут 5-6 мелких игроков. Коммерческий intent приличный.
- **B «one-time / no subscription»** — выдача это листиклы (moosend, themeisle, titan) + Zoho (free) + Quora. Прямых one-time сервисов почти нет (ближе всех Purelymail $10/год). Гэп есть, но SERP = контент, не сервисы → вероятно низкий объём. Проверить в GKP.
- **D «cloudflare email routing send as gmail»** — топ это DIY-гайды + Cloudflare community/docs + github. Intent = «сделать БЕСПЛАТНО самому». Ранкабельно, но монетизация низкая (они не платят).
- **E «multiple domains one gmail»** — блоги + Gmail Help + Salesforce. Информационный/DIY intent, низкая покупательская готовность.

## GKP — что снимать и критерии решения

Снимать по каждому ключу: **Avg monthly searches**, **Competition** (это рекламная, не SEO), **Top-of-page bid** (прокси денег в теме), **тренд**. Гео: US/UK/CA/AU + English.

- **GO:** в кластерах B/C/E есть ключи с объёмом сотни–тысячи/мес И ненулевым бидом.
- **KILL:** коммерческий лонг-тейл (one-time/no-sub/multiple-domain) = единицы-десятки/мес → спроса под угол нет.
- Головные (A) — high volume, но забиты гигантами → для валидации спроса, не для ранка.

## Честный итог SERP-чека

Ранкабельность на B/C/E — есть (мелкие игроки, не сплошь гиганты). НО два неудобных факта:
1. Коммерческий «without Workspace» кластер **плотнее**, чем думали — 5-6 мелких конкурентов + SendMailAs, не пустое поле.
2. Самые ранкабельные кластеры (D/E) — **free/DIY intent** → низкая готовность платить. Чисто-коммерческий и недо-занятый — только «one-time», но у него, вероятно, тонкий объём (решает GKP).

## Углы, куда НАШЕ SEO реально дотянется (новый низко-авторитетный сайт)

Ранжировано по winnability (слабость текущей выдачи), с оговорками. «Дотянется» = SERP слабый, НЕ = большой объём.

1. **One-time / no-subscription** — в выдаче НЕТ ни одного настоящего one-time сервиса (только листиклы + Zoho-free + Purelymail-annual). MailKit БУКВАЛЬНО и есть one-time-ответ → можно встать #1 как сам продукт под эти ключи. Максимальный фит + минимум сервис-конкуренции. Оговорка: объём, вероятно, маленький — решает GKP.

2. **Cloudflare + «done-for-you / automated / setup service»** — SERP по «cloudflare send as gmail» = DIY-блоги + форумы (слабо). Страница под «cloudflare email routing gmail — сделаем за тебя / one-click / setup service» ловит CF-юзера, который УЖЕ на DIY-гайде, но не хочет возиться и создавать токен. Точный тех-фит, обгоняемо против блогов. НЕ целься в чисто-free «how to free» — там не платят.

3. **Comparison / alternative страницы** — `sendmailas alternative`, `google workspace alternative just email`, `purelymail alternative`, `improvmx alternative send`. Bottom-funnel, высокий intent, и новые сайты РЕАЛЬНО ранкают comparison-страницы (специфика + свежесть). У тебя уже есть /compare — расширить в per-competitor alt-страницы. Оговорка: зависит от того, есть ли у конкурента name-search спрос (GKP).

4. **«without workspace» ультра-суб-лонгтейлы** — головной «without workspace» плотный (5-6 тулов), но узкие варианты (`...for freelancers`, `...for multiple domains`, `...cloudflare`, `...cheapest`) могут быть открыты.

**Общие оговорки:**
- Reachable ≠ high-volume. №1 (one-time) — самый winnable, но может быть тонким по объёму → GKP решает, стоит ли.
- №3 (comparison) — сильнейший рычаг для нового сайта (bottom-funnel + winnable), но нужен name-search спрос по конкурентам.
- Избегать чисто free/DIY-ключей (кроме реврайта в «done-for-you») — иначе ранкаешь тех, кто не платит.
